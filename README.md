# 協同準則 AI 學習系統 V5.1 自行選題版

新增功能：
- 首頁新增「自行選題」。
- 可切換全部、陸綱、指參、情報、聯兵營。
- 可勾選任意題目。
- 可一次勾選或取消目前科目。
- 可依勾選順序開始，或隨機排列後開始。
- 保留 V5.0 的練習、模擬考、逐字批改、錯題庫、收藏、搜尋、統計與方塊消除功能。

部署方式：
1. 解壓縮 ZIP。
2. 將 index.html、manifest.json、sw.js、README.md 全部覆蓋上傳到 GitHub quizapp 根目錄。
3. 等待 GitHub Pages 重新部署。
4. 若手機仍顯示舊版，請重新整理或移除主畫面舊圖示後再加入。
